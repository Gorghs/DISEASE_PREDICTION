Step 4: Create Render Web Service
   URL: https://render.com
   Type: Web Service
   Repository: Select your GitHub repo
   
   Build Command:
   pip install -r requirements.txt && python train.py
   
   Start Command:
   gunicorn --workers 1 --timeout 120 --bind 0.0.0.0:$PORT app:app
   
   Environment Variables:
   KAGGLE_USERNAME = [your-kaggle-username]
   KAGGLE_KEY = [your-kaggle-api-key]

Step 5: Deploy
   Click "Deploy" button on Render
   Wait ~1 hour for training to complete

Step 6: Access Your App
   URL: https://[your-app-name].onrender.com
   Upload image → Get diagnosis → Share!
