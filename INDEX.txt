What is this app?
  AI-powered banana leaf disease detector
  → Runs on web browser
  → User uploads image
  → AI diagnoses disease
  → Shows treatment tips

How does it work?
  1. User uploads banana leaf image
  2. Server validates image
  3. ResNet50 model analyzes image
  4. Returns disease diagnosis
  5. Shows treatment recommendations

Where does it run?
  On Render cloud (free tier)

  → In-memory model caching
  → Fast predictions

How much does it cost?

  Total: ~1.5 hours first time

Problems?

Model training takes too long?
  → Reduce EPOCHS in train.py to 5

Out of memory error?
  → Reduce BATCH_SIZE in train.py to 8

Kaggle download fails?
  → Verify username and API key in Render

App loads but predictions don't work?
  → Wait 2-3 seconds for model to load

Predictions very slow?
  → Upgrade Render plan for dedicated CPU

Can't access Render dashboard?
  → Check internet connection
  → Try incognito mode
